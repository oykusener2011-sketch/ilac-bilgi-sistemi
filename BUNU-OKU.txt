ADIM 1: EmailJS Hesabı Aç (2 dakika)
───────────────────────────────────────────────────────────

2. Sağ üst köşede "Sign Up" tıkla

4. Gmail hesabını seç: otacilar45.aifanin.kare.kodu@gmail.com

───────────────────────────────────────────────────────────
ADIM 2: Kurulum Web Sayfasını Aç (3 dakika)
───────────────────────────────────────────────────────────

1. Web sitesini aç:
   http://localhost:8000

2. Anasayfada aşağı kaydır → "İletişim" bölümünü bul

3. "⚙️ İlk Kullanımda: Kurulum Yap" linkine tıkla

4. Açılan sayfada bu 3 kodu EmailJS'den kopyala + yapıştır:

   1️⃣ PUBLIC KEY
      → EmailJS Dashboard
      → Sol menü: Account
      → Public Key bölümüne bak
      → Kodu kopyala (o7Uyw... şeklinde başlayan)
      → Web sitesine yapıştır

   2️⃣ SERVICE ID
      → EmailJS Dashboard
      → Sol menü: Email Services
      → Gmail hizmetini seç
      → Service ID kopyala (service_abc... şeklinde)
      → Web sitesine yapıştır

   3️⃣ TEMPLATE ID
      → EmailJS Dashboard
      → Sol menü: Email Templates
      → Yeni template oluştur:
         * Adı: template_ilac_mesaj
         * Service: Gmail seç
         * Kaydet
      → Template ID'ni kopyala (template_xyz... şeklinde)
      → Web sitesine yapıştır

5. "KAYDET VE TAMAMLA" butonu tıkla

✅ BITTI! Kodlar kaydedildi.

───────────────────────────────────────────────────────────
ADIM 3: Test (2 dakika)
───────────────────────────────────────────────────────────
