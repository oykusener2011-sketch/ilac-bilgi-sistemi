🚀 İlaç Bilgi Sistemi - Kurulum Rehberi

Hızlı Başlangıç Kılavuzu

👋 Hoş Geldiniz! Bu rehber, İlaç Bilgi Sistemini kendi bilgisayarınızda çalıştırmanız için adım adım talimatlar içermektedir.

📋 Sistem Gereksinimleri

Bileşen Gereksinim Not
İşletim Sistemi Windows, macOS, Linux Tamamı desteklenir
Web Tarayıcı Chrome, Firefox, Safari, Edge (Son 2 sürüm) Sesli okuma için uyumlu olanlar
İnternet QRCode.js kütüphanesi için gerekli Veya kütüphaneyi lokal olarak yükleyin
Kod Editörü İsteğe bağlı VS Code, Sublime Text vb.

⚙️ Kurulum Adımları

Adım 1: Dosyaları İndirin

Projenin tüm dosyalarını aynı klasöre kaydedin:

  • index.html
  • pharmacist.html
  • mobile.html
  • styles.css
  • script.js
  • pharmacist.js
  • mobile.js

Adım 2: Web Sunucusu Başlatın

Seçenek 1: Python ile (3.x sürüm)

cd /path/to/folder python -m http.server 8000

Seçenek 2: Node.js ile

npm install -g http-server http-server -p 8000

Seçenek 3: PHP ile

cd /path/to/folder php -S localhost:8000
💡 Not: Dosyaları doğrudan tarayıcıda açmak (file://) sesli QR kod gibi bazı özellikleri devre dışı bırakabilir. Web sunucusu kullanın.

Adım 3: Tarayıcıda Açın

Web tarayıcınızda aşağıdaki adresi yazın:

http://localhost:8000
✅ İlaç Bilgi Sistemi anasayfası görüntülenmelidir.

🧪 Test Etme

Eczacı Panelini Test Edin

1. Eczacı Paneli'ne gidin

2. Aşağıdaki test verilerini girin:

  • Hasta Adı: Ahmet Kaya
  • İlaç Adı: Aspirin 500mg
  • Amacı: Baş ağrısı ve ateş için
  • Doz: 500 mg
  • Sıklık: 2 (günde 2 kez)
  • Süre: 3 gün

3. "QR Kod Oluştur" düğmesine tıklayın

4. QR kodu indirin veya yazdırın

Mobil Tarayıcıyı Test Edin

1. Mobil Tarayıcı'ya gidin

2. Sayfa alt kısmında "🧪 Test Verisi Yükle" düğmesine tıklayın

3. İlaç bilgileri görüntülenecektir

4. "🔊 Tümünü Canlı Oku" düğmesiyle sesli okumayı deneyin

🔊 Sesli Okuma Ayarları

Windows:

  • Chrome, Edge, Firefox destekler
  • Sistem sesi açık olmalıdır
  • Dil ayarları: Türkçe (tr-TR)

macOS:

  • Safari, Chrome, Firefox destekler
  • Sistem tercihleri > Erişilebilirlik > Konuşma

Linux:

  • Chrome/Chromium kullanın
  • Espeak veya Festival kurulu olmalı

❓ Sıkça Sorulan Sorular

S: Sesli okuma çalışmıyor, ne yapmalıyım?

C: Tarayıcınızın ayarlarını kontrol edin:

  • Sistem sesinin açık olduğundan emin olun
  • Tarayıcı için sesli izin verildiğinden emin olun
  • Web Speech API'sini destekleyen bir tarayıcı kullanın
  • Başka bir tarayıcı deneyin

S: QR kod taraması çalışmıyor?

C: Test verilerini kullanabilirsiniz:

  • Mobil sayfasında "🧪 Test Verisi Yükle" düğmesini tıklayın
  • Veya eczacı panelinde oluşturduğunuz QR kodu manuel olarak girin

S: Dosyaları nereye koymalıyım?

C: Tüm dosyaları aynı klasörede saklayın. Örnek:

MyProject/ ├── index.html ├── pharmacist.html ├── mobile.html ├── styles.css ├── script.js ├── pharmacist.js └── mobile.js

S: Yerel ağda diğer cihazlardan erişebilir miyim?

C: Evet! Web sunucusunu başlatırken:

python -m http.server 8000 --bind 0.0.0.0
Daha sonra diğer cihazlardan http://[bilgisayar-ip]:8000 ile erişin

📱 Mobil Cihazda Kullanım

Mobil cihazlarda kullanmak için:

  1. Web sunucusunu başlatın
  2. Mobil cihazda bilgisayarınızın IP adresini girin: http://[IP]:8000
  3. İçeriyi Giriş Ekranına ekleyin (PWA olarak)
  4. Çevrimdışı da kullanılabilir

🐛 Sorun Giderme

Sayfalar yüklenmiyor:

  • Web sunucusunun çalışıp çalışmadığını kontrol edin
  • Doğru port numarasını kullanıp kullanmadığınızı kontrol edin
  • Dosya adlarını (büyük/küçük harf) kontrol edin

Stil yüklenmiyorsa:

  • styles.css dosyasının aynı klasörde olduğunu kontrol edin
  • Tarayıcı önbelleğini temizleyin (Ctrl+Shift+Delete)

QR kod oluşturulmuyor:

  • İnternet bağlantısını kontrol edin (QRCode.js CDN için)
  • Tarayıcı konsolunda hata olup olmadığını kontrol edin (F12)
